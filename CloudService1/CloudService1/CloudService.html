Windows Azure Tools For Microsoft Visual Studio

若要启用对在 Windows Azure 上构建可伸缩的 Web 应用程序和服务的支持，请安装 Windows Azure Tools for Microsoft Visual Studio。

Windows Azure Tools for Microsoft Visual Studio 可扩展 Visual Studio， 以便允许在 Windows Azure 上创建、生成、调试、运行和打包 可伸缩的 Web 应用程序和服务。安装内容包括 Windows Azure SDK。

请参见下载页面 了解有关系统要求、支持的操作系统和安装说明。

注意: 包含此页的项目仅提供安装必需工具集的说明。
